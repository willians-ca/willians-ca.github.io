Recicla una lata.
Gana una recompensa.

EcoRecompensa es una máquina automática de reciclaje de latas controlada por un Arduino Nano y una Raspberry Pi 5. El sistema utiliza neumática, sensores ultrasónicos, relés y comunicación serial para detectar, aplastar y contabilizar latas recicladas, generando automáticamente códigos de recompensa mediante una interfaz desarrollada en Python.

Proceso

¿Cómo funciona?

El sistema automatiza el reciclaje mediante neumática, sensores ultrasónicos, relés y comunicación serial entre Arduino Nano y Raspberry Pi 5.

PASO 01
🟢

Inicio del sistema

El usuario presiona el botón START y el Arduino Nano inicia automáticamente la secuencia del sistema neumático. También existe un botón rojo de parada de emergencia para detener todo el proceso inmediatamente.

PASO 02
💨

Empuje de la lata

El Arduino activa el Relé 1 para accionar una válvula neumática 3/2 que mueve el cilindro neumático 1, empujando la lata hacia una zona inferior. Luego el Relé 3 devuelve el cilindro a su posición inicial.

PASO 03
📡

Detección y aplastado

Un sensor ultrasónico detecta la presencia de la lata y el Arduino activa el Relé 2, controlando una válvula 5/2 que distribuye aire comprimido hacia dos cilindros neumáticos que aplastan la lata simultáneamente.

PASO 04
🖥️

Registro y recompensa

Cada vez que una lata es reciclada, el Arduino envía la palabra "RECICLADA" mediante comunicación serial a la Raspberry Pi 5, donde una interfaz en Python contabiliza las latas recicladas y genera códigos de recompensa automáticamente.

Hardware & Software

Tecnología integrada

EcoRecompensa integra control electrónico, automatización neumática y comunicación serial entre Arduino Nano y Raspberry Pi 5 para automatizar completamente el proceso de reciclaje y generación de recompensas.

  • 🔌
    Arduino NanoControla los relés, botones START y STOP, sensores ultrasónicos y toda la secuencia neumática del sistema.
  • 💨
    Minicilindros neumáticosEl cilindro neumático 1 desplaza la lata y los cilindros 2 y 3 trabajan simultáneamente para aplastarla mediante aire comprimido.
  • 📡
    Sensor ultrasónicoDetecta la presencia de la lata y permite que el sistema funcione automáticamente en bucle mientras existan latas por reciclar.
  • 🖥️
    Raspberry Pi 5Lee la comunicación serial del Arduino y ejecuta la interfaz gráfica de recompensas.
  • 🐍
    Software en PythonLa interfaz desarrollada en Python lee el monitor serial, contabiliza las latas recicladas y genera códigos de recompensa automáticamente.

Institución

Ingeniería Mecatrónica — UNAJ

Universidad Nacional de Juliaca (UNAJ) Juliaca, Puno — Perú

La Carrera Profesional de Ingeniería Mecatrónica de la UNAJ forma profesionales con sólidas competencias en electrónica, mecánica, sistemas de control y programación embebida.

EcoRecompensa nace como un proyecto de automatización aplicado al reciclaje, integrando electrónica, neumática industrial, programación embebida y desarrollo de interfaces gráficas para ofrecer una solución tecnológica orientada al cuidado ambiental.

El proyecto integra disciplinas transversales de la mecatrónica: control automático, electrónica digital, neumática y desarrollo de software, demostrando la capacidad formativa de la carrera.
